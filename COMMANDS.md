# Voice Commands Reference

Complete list of voice commands for the voice-controlled Gmail app.

## Inbox Navigation

| Command | Variations | Description |
|---------|------------|-------------|
| `show inbox` | "show my inbox", "open inbox", "go to inbox" | Display inbox emails |
| `show unread` | "show unread emails", "new emails", "unread" | Show only unread emails |
| `show sent` | "sent mail", "sent emails" | Show sent folder |
| `show drafts` | "my drafts" | Show drafts folder |
| `show starred` | "starred emails" | Show starred emails |
| `show spam` | "spam folder" | Show spam folder |
| `show trash` | "trash", "deleted emails" | Show trash folder |
| `check inbox` | "how many emails", "email count" | Get inbox/unread count |
| `refresh` | "refresh inbox", "check for new" | Reload current folder |

## Email Selection & Reading

| Command | Variations | Description |
|---------|------------|-------------|
| `open email <N>` | "read email <N>", "show email <N>", "email <N>" | Open email by number |
| `next` | "next email", "next one" | Open next email in list |
| `previous` | "previous email", "go back", "last one" | Open previous email |
| `first email` | "go to first" | Open first email in list |
| `last email` | "go to last" | Open last email in list |

## Email Actions

| Command | Variations | Description |
|---------|------------|-------------|
| `delete` | "delete this", "delete email", "trash this", "trash it" | Delete current email |
| `delete email <N>` | "delete <N>", "trash email <N>" | Delete email by number |
| `archive` | "archive this", "archive email", "done with this" | Archive current email |
| `archive email <N>` | "archive <N>" | Archive email by number |
| `mark read` | "mark as read" | Mark current email as read |
| `mark unread` | "mark as unread" | Mark current email as unread |
| `star` | "star this", "star email" | Star current email |
| `unstar` | "unstar this", "remove star" | Remove star from email |

## Labels

| Command | Variations | Description |
|---------|------------|-------------|
| `label <name>` | "label this <name>", "add label <name>" | Add label to current email |
| `remove label <name>` | "unlabel <name>" | Remove label from email |
| `show labels` | "list labels", "my labels" | List all available labels |
| `show label <name>` | "open label <name>" | Show emails with label |

## Search

| Command | Variations | Description |
|---------|------------|-------------|
| `search <query>` | "find <query>", "search for <query>" | Search emails |
| `from <name>` | "emails from <name>", "show from <name>" | Search by sender |
| `to <name>` | "emails to <name>" | Search by recipient |
| `subject <text>` | "about <text>" | Search by subject |

## Composing Email

| Command | Variations | Description |
|---------|------------|-------------|
| `compose` | "new email", "write email" | Start new email |
| `email <name>` | "send email to <name>", "write to <name>" | Start email to contact |
| `reply` | "reply to this", "respond" | Reply to current email |
| `reply all` | "reply to all" | Reply to all recipients |
| `forward` | "forward this", "forward email" | Forward current email |

## Draft Editing

| Command | Variations | Description |
|---------|------------|-------------|
| `subject <text>` | "subject is <text>" | Set email subject |
| `message <text>` | "body <text>", "say <text>" | Set/add to message body |
| `continue <text>` | "add <text>", "also say <text>" | Append to message body |
| `send` | "send it", "send email" | Send the draft |
| `cancel` | "cancel email", "discard", "nevermind" | Discard draft |
| `show draft` | "read draft", "what did I write" | Read back current draft |

## Attachments

| Command | Variations | Description |
|---------|------------|-------------|
| `open attachment` | "show attachment", "view attachment" | Open first attachment |
| `open attachment <N>` | "attachment <N>" | Open attachment by number |
| `open pdf` | "show pdf", "view pdf" | Open first PDF attachment |
| `download attachment` | "save attachment" | Download attachment |

## PDF Viewer

| Command | Variations | Description |
|---------|------------|-------------|
| `scroll down` | "down", "go down" | Scroll PDF down |
| `scroll up` | "up", "go up" | Scroll PDF up |
| `page <N>` | "go to page <N>", "page number <N>" | Jump to page |
| `next page` | "page down" | Go to next page |
| `previous page` | "page up" | Go to previous page |
| `first page` | "go to start", "beginning" | Go to first page |
| `last page` | "go to end", "end" | Go to last page |
| `zoom in` | "bigger" | Zoom in |
| `zoom out` | "smaller" | Zoom out |
| `close` | "close pdf", "back", "exit" | Close PDF viewer |

## Contacts

| Command | Variations | Description |
|---------|------------|-------------|
| `show contacts` | "list contacts", "my contacts" | List all contacts |
| `find contact <name>` | "search contact <name>" | Search for contact |
| `add contact <name> <email>` | "save contact" | Add new contact |
| `save sender` | "add sender to contacts" | Save current email's sender |
| `remove contact <name>` | "delete contact <name>" | Remove contact |

## Pagination

| Command | Variations | Description |
|---------|------------|-------------|
| `more` | "show more", "next page", "more emails" | Load more emails |
| `back` | "previous page", "go back" | Go to previous page |

## System

| Command | Variations | Description |
|---------|------------|-------------|
| `stop` | "stop listening" | End conversation mode |
| `help` | "what can I say", "commands" | List available commands |
| `repeat` | "say again", "what" | Repeat last response |
